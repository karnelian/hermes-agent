# makeskill

Hermes가 대화 중 생성하거나 정리한 Markdown 초안, 스킬 후보, AI 작업 운영 노트를 모아두는 공간이다.

이 폴더의 문서는 기본적으로 공식 제품 문서가 아니라 **정리/초안/후보 자료**다. 내용이 안정되거나 목적이 분명해지면 아래 위치로 승격할 수 있다.

- 공식 사용자/개발자 문서: `website/docs/`
- Hermes 재사용 스킬: `skills/`
- 프로젝트별 규칙 문서: `docs/rules/`
- 특정 작업 실행 계획: `docs/plans/`

## 운영 규칙

- Hermes가 새로 만드는 일반 Markdown 정리는 먼저 이 폴더에 둔다.
- 서로 관련 있는 내용은 가능한 한 기존 파일에 합쳐서 정리한다.
- 새 파일을 만들 때는 파일명이 주제를 명확히 설명해야 한다.
- 공식 문서나 실제 스킬로 승격하기 전에는 중복, 범위, 적용 대상을 다시 검토한다.

## 현재 문서

- `game-genre-taxonomy.md` — 게임 장르 분류/정리 초안
